README

My Fitness Application
This site is a catalog of Excercises sorted by muscle groups
Exercises have a description and video instructions

User must be signed in using Google plus to create, edit, or delete an item

API's
- http://localhost:8000/api/musclegroup
- http://localhost:8000/api/musclegroup/<musclegroup-id>

Requirements
-python 2.7
-CSS
-HTML
-Bootstrap 4
-Google Oauth2 api

Getting Started
1) Unzip charo_catalog_app.zip
2) run command below to install required python library's:
   $ pip install -r requirements.txt
2) Start the web application with:
    - catalog/python web_application.py
3) web application url: http://localhost:8000/
